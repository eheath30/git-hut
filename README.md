# Git Hut
### An github repo search engine
Search and favourite repos by github username.

### Live site at: `https://git-hut.netlify.app/`

##### This project was built using ReactJS


## Installation & Usage
### Installation
- cd into git-hut
- run: `npm i`

### Usage
- cd into git-hut
- run: `npm run start`
